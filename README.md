# AmazonPriceTracker
It's an app that can track price of the user specified commodity on Amazon.
If the price drop down, user will receive a notification from App.

Demo Link:
http://www.price-trackers.com/

How to run the code:
<pre><code>
start app:
forever start -c 'npm start' ./
stop app:
forever stopall
 </code></pre>
